# codedge/eloquence
An improved version of [Jarek Tkaczyk](https://github.com/jarektkaczyk/eloquence) extension of Eloquent ORM.

## Versions
Current version is in branch `5.3` and/or `master`.

For older versions of Illuminate/Laravel please use:
- 5.2.* -> [5.2](https://github.com/jarektkaczyk/eloquence/tree/5.2) branch.
- 5.1.* -> [5.1](https://github.com/jarektkaczyk/eloquence/tree/5.1) branch.
- 5.0.* -> [0.4](https://github.com/jarektkaczyk/eloquence/tree/0.4) branch.

## Improvement
* Mapping of relations foreign keys with `Mappable` extension
  * `hasOne` relation

## Documentation
Please find the [documentation](https://github.com/jarektkaczyk/eloquence/wiki) in the wiki of the [Sofa\Eloquence package](https://github.com/jarektkaczyk/eloquence).
